# parcel-start-kit

Setting up a professional project with Parcel as a code bundler.

## Feater

* ES6 with Bebel
* SCSS
* CSS Modules
* Typography.js
* Build for production


## Git Clone

```git clone https://github.com/sungkuk5420/parcel-start-kit.git```

## NPM Install

```npm install```

## Start

```npm start```

## Build

```npm run-script build```
