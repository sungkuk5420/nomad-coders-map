## 노마드 코더들은 어디에 사는가 
노마드 코더의 해외 근무자들의 위치를 지도로 표시하였습니다.

## 데모 사이트

https://sungkuk5420.github.io/nomad-coders-map/

## Git Clone

```git clone https://github.com/sungkuk5420/nomad-coders-map.git```

## NPM Install

```npm install```

## Start

```npm start```

## Build

```npm run-script build```

## 기능
 - 사용자 검색기능
 - 검색한 사용자 기반으로 도시표시
 - 도시의 거주 인원 수 표시
 - 핀 클릭시 해당 도시 거주자
